Introduction

Welcome

Aleph is a learning project, also used to develop plugins, originally based on Unreal Engine 4.26. Aleph is packed with C++ code and Blueprints to help you kickstart your game project.


This documentation represents Aleph & it's plugins as they function on the current build based on Unreal Engine 5.0.1!

Subtitles

Aleph uses the SubX plugin to display the subtitles. All you have to do to use it is call the "PlaySubtitles" in any blueprints that you'd like to play the subtitle and that's it.


Please make sure that the Subtitle widget is added to your hud before Playing the subtitle.
